# Credit System User Guide

## Welcome to Altered!  

Altered gives you **6 free credits** to try the AI-powered ADHD assistant. Once you've used them, you can continue unlimited by adding your own API key.

---

## How Credits Work

### Free Credits
- Every new user starts with **6 free credits**
- Each AI interaction uses **1 credit**
- Credits are tracked automatically
- No expiration date

### Checking Your Balance
1. Open **Settings** (tap the profile icon)
2. Look for the "Usage & Credits" section
3. Your current balance is displayed with a ⚡ icon

**Color Indicators:**
- 🟢 **Green (3-6 credits)**: You're all set!
- 🟡 **Yellow (1-2 credits)**: Running low
- 🔴 **Red (0 credits)**: Time to add your API key

---

## What Happens When Credits Run Out?

When your 6 free credits are used up, you'll see this message:

> ⚠️ **You've used all 6 free credits!**
>
> To continue using Altered, please add your own Gemini API key in Settings.
>
> Get your free API key at: https://makersuite.google.com/app/apikey

Don't worry! Adding your own key is quick and free.

---

## How to Add Your Own API Key (BYOK)

"Bring Your Own Key" (BYOK) gives you **unlimited usage** of Altered.

### Step 1: Get a Free Gemini API Key
1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the key (starts with `AIza...`)

### Step 2: Add Key to Altered
1. Open **Settings** in Altered
2. Scroll to "API Configuration"
3. Enter your API key in the text field
4. Tap **Save API Key**

Done! 🎉 You now have unlimited usage.

## FAQ

### How much does a Gemini API key cost?
Google's Gemini API has a generous **free tier**:
- 15 requests per minute
- 1 million tokens per day  
- More than enough for personal use!

[Check current pricing](https://ai.google.dev/pricing)

### Is my API key safe?
Yes! Your API key is:
- ✅ Encrypted at rest
- ✅ Never shared or logged
- ✅ Stored securely in Firebase
- ✅ Only used for your requests

### Can I remove my API key later?
Absolutely! Go to Settings → API Configuration → "Remove Custom Key"

### What if I run out of free credits without adding a key?
The app will stop responding to chat messages until you add your own API key. All your data and settings are safe - just add a key to continue!

### Can I get more free credits?
Currently, the 6 free credits are a one-time allocation. We may add monthly refills or paid tiers in the future!

## Technical Details

### Credit Consumption
- **1 credit** = 1 AI interaction
  - Chatting with the assistant
  - Creating calendar events
  - Task atomization
  - Any agent tool use

### BYOK vs Free Credits
When you have your own API key:
- ✅ Credits are NOT consumed
- ✅ Unlimited usage (subject to Google's limits)
- ✅ Billing goes to your Google account
- ✅ You control your usage and costs

Without a custom key:
- Uses organization's shared credits
- Limited to 6 free credits
- No billing to you

## Support

Having issues? Here's what to check:

### "API key validation failed"
- Make sure you copied the entire key (starts with `AIza`)
- Check there are no extra spaces
- Try generating a new key

### "Credits not updating"
- Pull down to refresh settings
- Log out and back in
- Contact support if issue persists

### "0 credits but I just signed up"
- Credit initialization may take a moment
- Restart the app
- Check your internet connection
